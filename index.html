Generating Rationale in Visual Question Answering (VQA)

A multimodal vision-and-language model that generates rationale and improve upon VQA task. Given a high order cognitive multiple choice question based on an image, a question and a set of four multiple choice options, the model must select the correct answer and generate the rationale behind choosing that answer. We used pretrained Vision-and-Language BERT (ViLBERT) to learn joint image and language representation to choose the correct answer and GPT-2 for rationale generation. Results show that joint training of these two modules helps to improve the performance in VQA task.
